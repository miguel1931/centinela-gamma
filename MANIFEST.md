## Información Técnica

### Tamaño Total del Paquete
- **Estimado**: ~120 MB (incluyendo dependencias)
- **Código fuente**: ~15 MB
- **Documentación**: ~3 MB
- **Configuraciones**: ~2 MB

### Requisitos del Sistema
- **Python**: 3.7 o superior
- **Memoria RAM**: Mínimo 2GB, recomendado 4GB
- **Almacenamiento**: Mínimo 500MB, recomendado 2GB
- **Red**: Conexión a internet estable para Twitter API
- **Sistema Operativo**: Windows, Linux, macOS

### Dependencias Principales
- tweepy (Twitter API v2)
- pandas (análisis de datos)
- flask (servidor web)
- sqlite3 (base de datos)
- requests (comunicaciones HTTP)
- datetime, json, logging (librerías estándar)
